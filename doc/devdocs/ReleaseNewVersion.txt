Releasing a new version of Purple Pen.

1. Updated change list with information about the new version. 

2. Update credits page in the help with credits to anyone.

3. Update VersionNumber.cs in Purple Pen.

4. Update VersionNumber.cs in MapModel

5. Build MapModel (Debug and Release), and copy new versions over with copy_mapmodel.bat

6. In solution explorer, highlight setup project, select View/Properties Window (NOT Property Pages). Changes
version code to match, say Yes when prompted to update product code.

7. Build release build.

8. Copy and rename src\Setup\Release\purplepen-setup.exe and place in http://purplepen.golde.org/downloads

9. Update Home and Download page to link to new download. (Both the Download Now button and the list of all versions).

10. Update News page with news about the new version.

11. Upload a new change list document.

12. Change "latest_version.txt" on web server http://purplepen.golde.org/downloads

13. Create a new tag in BZR.

14. Push BZR up to LaunchPad.
