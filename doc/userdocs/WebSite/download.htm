Click the button above to download the latest released version of Purple Pen. Run it to install Purple Pen on your computer.

Requirements:

Purple Pen requires Windows XP SP2, Windows Vista, or Windows 7. On Windows XP, Purple Pen requires the Microsoft .NET Framework 2.0 to run. If you do not have this installed on your machine, the installer will automatically download and install it onto your machine. If you wish to install on a Windows XP machine that is not connected to the internet, you can manually download, copy, and install the Microsoft .NET Framework 2.0 from here.

(Users report that Windows 2000 and Windows Me can also run Purple Pen, if Windows is fully updated; however, this has not been tested, and may not work in future versions.)
