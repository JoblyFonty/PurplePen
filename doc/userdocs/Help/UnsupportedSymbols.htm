Unsupported OCAD Symbol Options

The following lists symbol options for OCAD symbols that are not supported by Purple Pen, and will not print or draw on the screen exactly as they do in OCAD. If your map contains one of these, a warning dialog will appear when you attempt to print courses.

Line Symbols

On the "Decrease" page, no options other than off are supported. The symbols will be displayed in their full size on the entire line. This option is not used in any standard orienteering symbol.

Text Symbols

On the "Paragraph" page, the text alignment of "Justified" is not supported. Justified text will appear left aligned in Purple Pen.

On the "Paragraph" page, the "Character spacing" option is not supported. Text will appears as though the character spacing were set to 0 (the default).

On the "Paragraph" page, the "Word spacing" option is not supported. Text will appear as though the word spacing were set to 100 (the default).

On the "Tab" page, no options are supported. Tabs will not appear in the text.

On the "Line Below" page, no options are supported. Any line below text will not appear.

On the "Framing" page, no options are supported. Text will appear unframed.

Line Text Symbols

On the "Spacing" page, the text alignment of "All line" is not supported. All-line text will appear left aligned in Purple Pen.

On the "Spacing" page, the "Character spacing" option is not supported. Text will appears as though the character spacing were set to 0 (the default).

On the "Spacing" page, the "Word spacing" option is not supported. Text will appear as though the word spacing were set to 100 (the default).

On the "Framing" page, no options are supported. Text will appear unframed.
