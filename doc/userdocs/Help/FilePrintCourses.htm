File/Print Courses...

Prints the courses and underlying OCAD map onto a printer. By default, the entire map will be printed. To chose the area that will be printed, use the File/Set Print Area command.

Note: Purple Pen prints almost all OCAD map symbols exactly as OCAD would print them. If Purple Pen encounters a symbol in the OCAD map that it cannot print correctly, then it will display a dialog listing the symbols that will be printed correctly.

 

This dialog has several options to customize the result of printing.

Output section

This section shows which printer will be used, and the paper size. Click Change Printer to change which printer to use. In that dialog, click Preferences to change the paper size and other printing options. The orientation of the paper (portrait or landscape) is chosen automatically to fit the best.

Courses section

This section shows which courses will be printed. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once.

Copies section

This section determines how many copies of each course will be printed.

Appearance section

Several things are set in this area:

Print area too large: This setting determines what happens if the print area is too large to fit on a single sheet of paper. Crop to a single page (the default) will pick the portion of the map that best covers that course. Print on multiple pages will print multiple pages of maps to cover the entire print area.

Color Model: Determines how the colors on the map are converted into colors on the printer:

  • OCAD Compatible: Picks the color model based on how OCAD does: the CMYK color model for PostScript and PDF printers, and the RGB color model for other printers. This setting should match OCAD behavior very closely.
  • RGB: Translates the Cyan-Magenta-Yellow-Black colors in the map to Red-Green-Blue colors using a simple formula. The colors on the printer will typically not match the intended colors exactly. This is the method used by versions of Purple Pen prior to 2.0.
  • CMYK: The default. Keeps the Cyan-Magenta-Yellow-Black colors as true as possible, using a standard color profile of CMYK colors. This generally produces the most accurate rendition of the intended colors.

Print Map Exchanges on Same Map: This setting only appears if a map exchange is present in the event. If this is checked, all parts of the course are printed together, overlapping. If unchecked, a separate map is printed for each part of the course. Typically this should be left unchecked unless you want to see all parts together for some reason.

Rasterize Before Printing: If checked, Purple Pen converts the map to an in-memory bitmap at the printer's resolution before sending the bitmap to the Windows printer driver. If unchecked, Purple Pen sends vector drawing commands to the Windows printer driver. If your print appears incorrect, printing doesn't work at all, or printing is extremely slow, checking this box may help. In most cases, leaving this unchecked should produce faster printing and smaller spool files. On Windows XP, or if the map is a bitmap or PDF, this box is always checked and cannot be changed.

Other buttons

Print: This button prints the courses on the selected printer. The dialog box closes.

Preview: Opens a print preview dialog box that shows what will be printed.

Cancel: Closes the dialog box without printing anything.
