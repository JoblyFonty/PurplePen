Course/Properties...

Change the properties of the current course. (If the All Controls map is displayed, then the All Controls Properties dialog is displayed instead.)

The following properties can be changed:

Course name: The name of the course. This name is displayed in the first box of the header row of the description sheet. The name should generally be short.

Course Type: The type of course. Allowable types are:

  • Normal course: A normal orienteering course with a fixed order of controls.
  • Score course: A score-O or freestyle orienteering course, with no fixed order of controls. No lines are drawn between the controls, and no length is computed. Optionally, a score can be associated with each control.

Climb: The climb, in meters, on the course. If left blank, no climb is displayed on the description sheet.

Class list/secondary title: The text entered here is displayed on the second line of the description sheet, below the event title and about the course name, length and climb. This line is often used to list the competitor classes using the course, but this line can be used for alternate course-specific information if needed.

Map printing scale: The scale that the course is to be printed at. Setting this correctly is important, because it affects the size of the control circles on the map, as well as the actual printed scale. Note that different courses can be printed at different scales (e.g., Brown at 1:10,000 and Blue at 1:15,000).

Description appearance: Controls how control description sheets appears. The choices are:

  • Symbols: the standard IOF symbolic description sheet, with 8 columns and symbols used to describe controls.
  • Text: instead of symbols, an English language textual description of the control is used. The textual description fills the area normally occupied by the last 6 columns of the description. The textual description is automatically generated.
  • Symbols and Text: The IOF symbols, with text equivalents to the right. The textual description for each control is automatically generated.
