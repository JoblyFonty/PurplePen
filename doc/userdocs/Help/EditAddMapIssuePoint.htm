Add/Map Issue Point

Adds a map issue point to the beginning of a course. The map issue point indicates the spot before the start triangle where maps are given to competitors. The map issue point is connected to the start triangle with a dashed line. The distance between the map issue point and the start is included in the calculated length of the course.

To have the dashed line fro the map issue point to the start follow a particular path, select that path and then use the Item/Add Bend command to add bending points to the line.
